# v0.3 task: snippet reuse workflow

Build a tiny local CLI in `workspace/` that reads a markdown file and prints TODO items as JSON.

Requirements:

- Use the snippets catalog as the primary starting point.
- Before implementation, identify one matching snippet and adapt it.
- Keep implementation minimal and local.
- Add a small test command or script to verify behavior.
- No OAuth, API keys, paid services, databases, or UI.

Acceptance criteria:

- `spec.md` and `interfaces.md` are generated.
- Snippet reuse is explicit in the implementation notes.
- `workspace/` has runnable code and simple verification command.
- `progress.md` records tested command/results.
